Redesign Binar Academy

UI/UX Batch10 Binar Academy

Overview

Binar Academy is an educational platform for young people who want to learn skills in technology through several programs such as Binar Bootcamp UI/UX Research & Design, Full Stack, Android, PM, and so on. In addition, there are also Binar Insights for skills support events to collaborate with digital talent search companies.

Objective

Make it easier for binar user to meet their information needs while studying through references, and become a one-stop learning app.

Solution

Applications that provide various information and needs of binar users to facilitate the learning process.

Persona

The goal of creating a user persona is to understand the user's needs and pain points.

Needs

  • Can find out class schedules and fill in attendance in the application 
  • Get class materials easily
  • Get interactive and engaging teaching
  • Get a reminder for the next class and the latest info from Binar
  • See a summary of class material for each session
  • Can see grades and reports for each challenge and quiz that is done

Pain Points

  • User Can't find the online class schedule and no reminder
  • The material in the application is not interesting
  • Unable to fill and view attendance
  • Confused in accessing reading material
  • The application does not provide convenience to the user's needs
  • There is no summary or report for each chapter

Understand

POV

User

Binar Academy students 20 -30 years old

Needs
